What I Use

HTML

When it comes to creating a websites, HTML is the one and only king for its layout. With a system that relies on opening and closing tags, I can quickly put down the foundaitons of a site

CSS

Without this, every site would look the same and boring. By using selectors, I can target elements and start adding selectors to give life to a website

JavaScript

Not to be confused with Java, JavaScript is what powers front-end interactivity with users. Using syntax similar to C++, code can be written to tell buttons what to do, when to trigger an animation, and so much more

Java

While not the best language when it comes to writting quick scripts, I still use Java to efficiently create software and games using JFX. On top of that, WPILIB has the largest amount of support for their Java version

Python

When it comes to needing to write a quick script or a task that runs during startup, Python is my goto language. Not only is it easier to start writting code, the built in package manager, PIP, allows for easy import of libraries
