chmod

Explination of numbers

Permission: 1 - can execute 2 - can write 4 - can read

3 (1+2) - can execute and write 6 (2+4) - can write and read

Position of the digit: 1 - what owner can 2 - what users in the file group can 3 - what users not in the file group can

chmod 700 - owner can read, write, and execute chmod 777 - everyone can read, write, and execute

shutdown

Explination shutdown [-akrhPHfFnc] [-t sec] time [message]

Flag Result
-r Reboot after shutdown
-P Shutdown and then power down
now Shutdown right after execution

sudo shutdown -r now
