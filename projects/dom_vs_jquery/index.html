JavaScript v. jQuery

What is the difference?

This page serves as a reference for me as I learn JavaScript. It outlines key differences between DOM and jQuery. DOM is JavaScripts built in link for connecting to HTML and CSS. At release, many people didn't like using the DOM API so jQuery was created in response. It allows for shorter and cleaner code but can slow down websites due to bloated content. Either way, knowing how to confidently use both is important to becoming a seasoned web developer

DOM

Targeting

JavaScript


// Old method
document.getElementById("my-id");
document.getElementsByTagName("my-name");
document.getElementsByClassName("my-class");

// New method
document.querySelector("div.class");
document.querySelectorAll("div.class");

jQuery


$('div.class');

Modifying CSS

JavaScript


var p = document.querySelectorAll("p");
for(var i = 0; i < p.length; i++) {
	p[i].style.color = "blue";
	p[i].style.border = "1px solid red";
}

jQuery


$('p').css({ 
	color: 'blue',
	border: '1px solid red'
});

Adding classes

JavaScript


.correct {
	color: green;
}

var questions = document.querySelectorAll("p.wrong");
for(var i = 0; i < questions.length; i++) {
	questions[i].classList.add("correct");
	questions[i].classList.remove("wrong");
	questions[i].classList.toggle("other");
}

jQuery


.correct {
	color: green;
}

$('p.wrong').addClass(".correct");
$('p.wrong').removeClass(".wrong");
$('p.correct').toggleClass("other")

Events

On click

JavaScript


var btn = document.querySelector('#btn-one');
btn.addEventListener("click", function() {
	this.style.border = "1px solid orange";
});

jQuery


$('#btn-one').on('click', function() {
	$(this).css("border", "1px solid orange");
});
